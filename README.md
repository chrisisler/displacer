# displacer
Rudiment permutation generator.

Input
```javascript
rlrr
```

Output
```javascript
[
    'RlrrLrll',
    'LrrlRllr',
    'RrlrLlrl',
    'RlrlLrlr',

    'LrllRlrr',
    'RllrLrrl',
    'LlrlRrlr',
    'LrlrRlrl'
]
```
